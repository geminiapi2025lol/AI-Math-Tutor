# AI Math Tutor

An AI-powered math tutoring application that helps students solve and understand mathematical problems across different levels of learning.

## Created By
**Daniel Kulov**  
Student at Yago International School, Seville, Spain  
Grade: 10th (MYP 5)  
Technology Class Project

## Features

- **Multiple Education Levels**: From Kindergarten to Further Mathematics
- **AI-Powered Tutoring**: Powered by Google Gemini 2.5 Flash
- **Interactive Chat Interface**: Ask questions and get step-by-step explanations
- **Example Problems**: Quick-start examples for each level
- **Two Beautiful Themes**: Modern and Classic designs

## Theme Switching

### How to Switch Between Themes

You can easily switch between two different visual designs:

1. **Modern Theme** (Default)
   - Clean, minimalist design
   - Soft colors and modern gradients
   - Generous white space
   - Smooth animations
   - Light gray background

2. **Classic Theme** (My Original Design)
   - Bold gradient backgrounds
   - Vibrant blue colors
   - Traditional card-based layout

## How to Use

1. Click "Run" to start the application
2. Close the welcome message by clicking "Okay"
3. Select your math level from the sidebar
4. Type your math question in the text box
5. Click "Send" and get AI-powered help!
